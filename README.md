# OmniLocker
- Removes the need to have items for crafting directly in your inventory, similar to Subnautica 2 EarlyAccess.
- Checks life pod containers, portable containers and various other container types.
- Adds crafting UI indicating when local containers will be used for building.

** Pre-prod release **
I made this for my own use, so I'm still testing it locally and it may contain bugs. Do not use this
mod if you are looking for something fully battletested! 

## Requirements
### Subnautica
This is only tested on the latest stable version of Subnautica from the Steam store.
### BepInEx
Mod loader for Unity games, [download it and use the installation instructions here](https://github.com/toebeann/BepInEx.Subnautica)

## Installation
1. Get the .zip file from the [releases page](https://github.com/TobyWilkes/OmniLocker/releases).
2. Extract the files
3. Copy the contents, including the OmniLocker folder into your Subnauticas BepInEx Plugins folder
   - On steam, find this by Right-clicking the game > Properties > Local Files > Browser
   - Open BepInEx/Plugins
   - Drop the extracted folder there
